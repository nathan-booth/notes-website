Lesson 7: Sets and Recursion

Sets

Python can be used to implement basic set theory concepts. Each element of a set is unique, which is not necessarily the case for lists or tuples. Elements of sets must be immutable but the sets themselves may be mutable or immutable (called frozen sets). The set function is not necessary to build a set; instead, braces may be used. The standard library includes functions for basic set operations including identification of a set, difference and symmetric difference, union, and intersection.

Recursion

A recursive function uses a limited version of itself within itself. A recursive definition requires at least two rules, a base case and a recursive case (or recursive call, in a function). It's the base case that eliminates circular logic or infinite recursive depth. Common recursive examples include fractals, Pascal's triangle, and the Fibonacci sequence. Data structures may be recursively defined, such nested number lists composed of either numbers or other nested number lists.
